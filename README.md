# Valentes na Fé — página de vendas

Projeto estático pronto para publicar no GitHub e conectar com a Vercel.

## O que foi atualizado

- A imagem principal do topo agora usa **uma única imagem** do álbum com figurinhas ao redor.
- A seção **Prévia interativa** foi ajustada para usar a nova capa em **9:16** sem cortar o título.
- As **setas da prévia** foram reposicionadas para **baixo da página**, com a **numeração no centro**.
- A vitrine de figurinhas foi reorganizada para ficar **mais limpa**, com as imagens melhor alinhadas e sem a borda branca anterior.
- A imagem da **área de membros** foi substituída pela nova versão com a mão segurando o celular.
- A imagem da seção final de oferta foi trocada pela nova composição com os álbuns empilhados.
- Foram removidas referências de linguagem ligadas a **futebol/time/campo** para deixar a página alinhada com o tema **Valentes na Fé**.

## Publicação no GitHub

1. Crie um repositório novo no GitHub.
2. Envie `index.html` e `README.md` para a raiz do projeto.
3. Faça o commit.

## Publicação na Vercel

1. Entre na Vercel.
2. Clique em **Add New Project**.
3. Importe o repositório do GitHub.
4. Escolha **Other** ou **Static HTML**.
5. Publique normalmente.

## Observação

As novas imagens principais usadas nas seções atualizadas foram incorporadas diretamente no `index.html`, facilitando a publicação em um projeto estático simples.
